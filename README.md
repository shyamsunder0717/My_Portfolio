# 🛡️ Shyam Sunder Sharma — Cybersecurity Portfolio

> **🌐 Live Website → [https://shyam07-portfolio.vercel.app/](https://shyam07-portfolio.vercel.app/)**

A high-end, responsive **3D Cybersecurity Portfolio** built with React, TypeScript, and Vite. Featuring immersive 3D animations, glassmorphism UI, and a sleek dark aesthetic — designed to showcase expertise in Cybersecurity, Penetration Testing, SOC Operations, and AI-driven development.

---

## ✨ Features

- 🎨 **3D Animations** — Three.js powered interactive background
- 💎 **Glassmorphism UI** — Modern frosted-glass design language
- ⚡ **Blazing Fast** — Built with Vite for instant HMR and optimized builds
- 📱 **Fully Responsive** — Works seamlessly across all devices
- 🌙 **Dark Mode** — Cyberpunk-inspired dark aesthetic
- 🎭 **Smooth Animations** — GSAP + Framer Motion transitions
- 🔗 **SEO Optimized** — Proper meta tags and semantic HTML

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/shyamsunder0717/My_Portfolio.git

# Navigate to project directory
cd My_Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```
